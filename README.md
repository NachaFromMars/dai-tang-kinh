# dai-tang-kinh — Buddhist canon reference, distilled and citable

> A Q&A reference skill for the Nikaya/Agama summaries, condensed into 100 pages. Ask a question, get an answer grounded in cited canonical sources.

[![OpenClaw Skill](https://img.shields.io/badge/OpenClaw-Skill-blueviolet)](https://github.com/NachaFromMars)

## Overview
dai-tang-kinh is a Buddhist canon reference skill built on *"Đại Tạng Kinh — Tóm tắt trong 100 trang"* by Hạt Bồ Đề (Gosinga). The material is organized into eight thematic reference files. When you ask a question, the skill identifies the relevant topic, reads the matching file, and answers with citations to specific sutta numbers and page references.

## Features
Eight reference files:
| File | Coverage |
|---|---|
| 01-gioi-thieu | Introduction, history, Tipitaka structure |
| 02-lich-su | Pali canon history, the Buddha, chief disciples |
| 03-muc-dich-kho-vui | Four Noble Truths, happiness, Tứ Thực |
| 04-giac-ngo | Enlightenment, jhana, rebirth, Tam Minh |
| 05-giao-ly | Sila-Samadhi-Panna, mindfulness, Tam Bảo |
| 06-tam-than | Nama-rupa, Five Aggregates, 12 Nidanas |
| 07-tu-tap | Paramitas, Four Foundations, Four Noble Fruits |
| 08-phu-luc | Appendix, pilgrimage, Temple Stay |

## Usage / Quick Start
Ask any question about Buddhist doctrine. The skill identifies the topic, reads the matching reference file, and responds with citations.
